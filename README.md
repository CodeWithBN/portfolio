UI & App Screens Gallery

This project is a clean and professional image gallery designed to upload, categorize, and display UI and application screen designs.
It focuses on clarity, usability, and real-world layout patterns commonly used in portfolios and internal tools.

Key Features

Category-based image upload (Mobile UI, Web UI, Dashboard, Auth Screens, Concepts)

Custom styled file upload with clear user feedback

Grid-based gallery with consistent image sizing

Filter buttons to quickly view specific categories

Empty state message when no images are available

Fully responsive layout

Built using plain HTML, CSS, and JavaScript (no libraries)

Purpose
This component can be used as:

A portfolio gallery section

A UI/UX showcase

A design reference organizer

A reusable frontend module for web projects

Tech Stack

HTML5

CSS3

JavaScript (Vanilla)
